# Images Directory

This directory contains images for the SaloneHub application.

## Required Images

- `icon-192.png` - 192x192px PWA icon
- `icon-512.png` - 512x512px PWA icon  
- `og-card.png` - 1200x630px Open Graph social sharing image
- `hero-bg.svg` - Hero section background illustration
- `sierra-leone-map.svg` - Animated Sierra Leone map illustration

## Image Guidelines

- Use the official color palette:
  - Primary: #0F4C5C (Deep teal)
  - Accent: #F4A261 (Gold)
  - Background: #F8FAFC (Clean white)
- Icons should be clean, modern, and professional
- Ensure accessibility with proper alt text
- Optimize for web (use WebP format when possible)
- Keep file sizes small for fast loading

## Notes

For the hackathon demo, you can use placeholder images or create simple SVG illustrations. The PWA icons are essential for app installation functionality.
